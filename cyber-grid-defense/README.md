# Cyber Grid Defense

A classic arcade-style defense game with a sleek, dark cyber aesthetic built purely with HTML5 Canvas, CSS3, and JavaScript.

## 🚀 Play the Game
To play the game, simply open the `index.html` file in any modern web browser. No complex build tools or dependencies are required.

## 🎮 Controls
* **Move Left:** `Arrow Left` or `A`
* **Move Right:** `Arrow Right` or `D`
* **Shoot:** `Spacebar`

## 👾 Features
* Dynamic wave scaling (enemies get faster and grid size increases).
* Multiple enemy types: Basic (Red), Fast (Orange), Armored (Purple).
* Particle explosion effects upon enemy destruction.
* Neon cyber-grid visual style.
