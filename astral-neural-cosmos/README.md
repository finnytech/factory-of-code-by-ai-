# Astral Neural Cosmos (Interactive Autonomous Galactic Civilization & AI Engine)

![License](https://img.shields.io/badge/License-MIT-brightgreen.svg)
![Status](https://img.shields.io/badge/Build-Passing-success.svg)

An autonomous AI ecosystem simulation and procedural cosmic audio-visual synthesizer engine. Built with vanilla JavaScript, HTML5 Canvas, Web Audio API, and custom QuadTree spatial acceleration.

---

## Key Features

- 🌌 **QuadTree Accelerated Physics**: Handles spatial partitioning for celestial gravitation, particle forces, and $O(N \log N)$ sensory field queries for hundreds of neural entities.
- 🧠 **Neuro-Genetic AI Evolution**: Multi-Layer Perceptron (MLP) neural brains per agent with real-time genetic algorithms (crossover, mutation, fitness selection, and lineage tracking).
- 🎵 **Generative Web Audio Synthesizer**: Procedural ambient soundscapes, chord modulations, LFO pulse shimmer, and real-time frequency spectrum visualizer.
- 🛸 **Cyberpunk Glassmorphism UI**: Live telemetry HUD, real-time FPS/population/fitness graphs, and interactive Neural Brain Inspector showing live synaptic node activations.
- ⚡ **Zero External Dependencies**: Pure vanilla JavaScript performance guarantee (< 60 FPS target).

## Getting Started

### Run Locally
```bash
npm start
# Or open index.html directly in any modern Web Browser (Chrome / Firefox / Edge / Safari)
```

### Run Automated Tests
```bash
npm test
```

---

## Project Architecture

- `index.html` - Semantic HTML5 layout with glassmorphism panels.
- `styles.css` - Custom cyberpunk theme, neon accents, and responsive layout.
- `js/engine.js` - Vector2D math, QuadTree spatial partitioning, ResourceNodes, CelestialBodies, and PhysicsEngine.
- `js/audio.js` - Web Audio API synthesizer drone & sound effect generators.
- `js/genetics.js` - Multi-Layer Perceptron Neural Networks & Agent evolutionary behavior.
- `js/ui.js` - Telemetry HUD manager, canvas charts, and Neural Brain Inspector drawer.
- `js/app.js` - Main simulation & visual render loop.
- `tests/test-suite.js` - Node.js automated test runner.
