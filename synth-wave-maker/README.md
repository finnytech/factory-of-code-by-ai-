# Synth-Wave-Maker

A web-based audio synthesizer and visualizer using the Web Audio API with a retro synth-wave aesthetic.

## Features

- Real-time sound generation using Web Audio API Oscillators.
- Adjustable waveform types: Sawtooth, Square, Sine, Triangle.
- Envelope controls: Attack and Release times.
- Master volume control.
- Interactive HTML/CSS generated piano keyboard playable via mouse or computer keyboard.
- Real-time audio visualization using Canvas.
